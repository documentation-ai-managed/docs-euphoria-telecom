## Prerequisites

<Callout kind="info" title="Before You Begin">
Ensure you have:
- A valid email address for account creation
- Business details ready (company name, address)
- Access to a web browser or the Euphoria Telecom mobile app
- An active internet connection for VoIP calls
</Callout>

## Create and Verify Your Account

Follow these steps to get your Euphoria Telecom account up and running quickly.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit the [Euphoria Telecom dashboard](https://dashboard.example.com) and click "Create Account".

    Fill in your business details:
    - Company name
    - Email address
    - Phone number for verification

    Submit the form to receive a verification email.
  </Step>

  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification link from Euphoria Telecom.

    Click the link to activate your account. If you don't see the email, check your spam folder.
  </Step>
</Steps>

## Assign Phone Numbers

Once verified, assign numbers to start using the service.

<Steps>
  <Step title="Browse Numbers" icon="phone">
    Log in to your dashboard at `https://dashboard.example.com`.

    Navigate to **Phone Numbers** > **Acquire Numbers**.

    Search for available numbers by area code or features (e.g., toll-free).
  </Step>

  <Step title="Assign Number via API" icon="code">
    For programmatic assignment, use the API:

    <CodeGroup tabs="JavaScript,cURL">
    ````javascript
    const response = await fetch('https://api.example.com/v1/phone-numbers', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        areaCode: '27',  // South Africa
        type: 'local'
      })
    });
    const number = await response.json();
    console.log('Assigned:', number.phoneNumber);
    ````

    ````bash
    curl -X POST https://api.example.com/v1/phone-numbers \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "areaCode": "27",
        "type": "local"
      }'
    ````
    </CodeGroup>
  </Step>
</Steps>

<ParamField path="phone-numbers" param-type="POST" required="true">
  Endpoint to acquire new phone numbers.
</ParamField>

<ParamField query="areaCode" param-type="string" required="true">
  Area code for the number (e.g., `27` for South Africa).
</ParamField>

## Configure Basic Call Settings

Set up your call routing and greetings.

<Tabs>
  <Tab title="Web Dashboard" icon="globe">
    In the dashboard:

    1. Go to **Call Settings** > **IVR Menus**.
    2. Create a menu: "Press 1 for Sales, 2 for Support".
    3. Assign your phone number to the menu.

    Save changes. Test with a quick preview call.
  </Tab>

  <Tab title="API Configuration" icon="api">
    Update settings programmatically:

    ````javascript
    await fetch('https://api.example.com/v1/call-settings', {
      method: 'PUT',
      headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
      body: JSON.stringify({
        phoneNumber: '+27123456789',
        ivrMenu: {
          prompt: 'Welcome to Euphoria Telecom. Press 1 for sales.',
          options: {
            '1': '+27123456789'
          }
        }
      })
    });
    ````
  </Tab>
</Tabs>

## Make Your First Test Call

<Callout kind="tip">
Use the built-in test caller in the dashboard to verify setup without external calls.
</Callout>

<Steps>
  <Step title="Initiate Test Call" icon="phone-call">
    In **Phone Numbers**, select your assigned number.

    Click **Test Call** and dial an internal extension or your mobile.
  </Step>

  <Step title="Monitor the Call" icon="activity">
    View real-time metrics in the **Call Monitor** tab:

    - Call duration
    - Agent performance
    - Route taken

    Confirm AI notes capture the summary.
  </Step>
</Steps>
